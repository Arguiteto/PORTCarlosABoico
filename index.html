Como fazer o tour

Essa ferramenta foi separada do sistema principal para você editar no GitHub. Ela funciona em HTML puro, com Pannellum por CDN para fotos 360°.

1. Organize as imagens

  • Crie uma pasta assets/ no repositório.
  • Coloque fotos 360° e sketches nela.
  • Use caminhos como assets/sala.jpg ou assets/planta.png.

2. Adicione ambientes

  • Escolha Foto 360° para imagens equiretangulares.
  • Escolha Sketch / foto comum para croquis, plantas, cortes, imagens de estudo ou moodboard.

3. Crie pontos

  • Em foto 360°, gire a câmera até o local desejado e clique em Marcar ponto.
  • Em sketch, clique em Marcar ponto e depois clique direto na imagem.
  • Escolha o destino para navegar entre ambientes.

4. Salve no GitHub

Clique em Baixar pacote GitHub. Ele gera um ZIP com o index.html, a pasta assets/ e instruções. Depois é só subir tudo no repositório.

tour-sketch-tool/ ├─ index.html └─ assets/ ├─ sala.jpg ├─ cozinha.jpg └─ sketch-planta.png
