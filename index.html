Mapoteca — Busca de terreno

Localizar arquivo do terreno

Informe o CEP do local. O sistema busca o endereço, geocodifica o ponto, tenta coletar bairro/lote/quadra/área no SIMGEO e procura a prancha correspondente na Mapoteca oficial.

Ajuda a posicionar o lote dentro do CEP.
Use só se souber; ajuda quando a Mapoteca usa nome antigo.

Observação: a Mapoteca pode publicar PDFs por bairro, loteamento, setor ou folha. Quando não houver API pública suficiente para confirmar o lote exato, o sistema mostra o arquivo mais provável e alternativas.

Use latitude/longitude em WGS84. Ex: Google Maps.
