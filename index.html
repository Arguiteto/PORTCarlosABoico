Processo

Escolhe, abre e navega.

O cliente entra pelo hall, escolhe uma capa no carrossel e abre a página do projeto com fotos, renders, informações e desenhos.

01

Hall de entrada

Carrossel horizontal corrigido, com setas, arraste no mouse/toque e rolagem pelo trackpad.

02

Página do projeto

Cada projeto abre uma tela interna com barra superior, logo como home, nome do projeto e botões Info e Desenhos.

03

Fotos e plantas

A galeria vertical passa automaticamente. Ao clicar, abre uma visualização horizontal ampla com setas laterais.

Sobre CB

Este espaço é para uma apresentação curta sobre Carlos A. Boico: área de atuação, estilo de trabalho, diferenciais e o tipo de projeto que deseja atrair.

A estrutura já está preparada para substituir os placeholders por fotos reais, renders e desenhos técnicos enviados para o sistema.
