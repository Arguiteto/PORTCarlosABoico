Como fazer

Modelo 01 · Planta baixa

  • Envie um print ou imagem da planta baixa.
  • Clique em “+ ambiente na planta” e marque cada cômodo.
  • Crie as conexões entre os cômodos que têm porta/passagem.
  • Informe yaw aproximado: 0 frente, 90 direita, -90 esquerda, 180 atrás.

Modelo 02 · Renders ou fotos 360°

  • Selecione um ambiente e aplique a imagem correspondente.
  • Use tipo “Foto 360° / render equiretangular” para imagens panorâmicas.
  • Abra o Tour, olhe para a passagem e use “Ponto pela vista” para ajustar o yaw/pitch.

Modelo 03 · Sketch / croqui

  • Use imagem comum quando ainda não tiver render 360°.
  • Os pontos usam posição X/Y da imagem ou a posição marcada na planta.

Modelo 04 · GitHub

  • Clique em “Baixar HTML pronto” para ter um arquivo único.
  • Ou clique em “Baixar pacote GitHub” para gerar zip com assets/ e README.
  • Suba no GitHub Pages e publique.
